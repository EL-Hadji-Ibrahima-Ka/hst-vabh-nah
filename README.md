# Mini Site Web

Un projet de mini site web simple et moderne créé avec HTML5 et CSS3.

## 📋 Description

Ce projet est un mini site web responsive qui présente une structure simple avec :
- Une page d'accueil élégante
- Une section "À propos"
- Une section "Contact"
- Un design moderne et responsive

## 🚀 Fonctionnalités

- **Design Responsive** : S'adapte parfaitement à tous les appareils (desktop, tablette, mobile)
- **Navigation Smooth** : Navigation fluide avec scrolling doux
- **Design Moderne** : Interface épurée avec des dégradés et des animations subtiles
- **Code Propre** : HTML5 sémantique et CSS3 moderne

## 📁 Structure du projet

```
mini-site-web/
├── index.html          # Page principale HTML
├── style.css          # Feuille de style CSS
└── README.md          # Documentation du projet
```

## 🛠️ Technologies utilisées

- **HTML5** : Structure sémantique du site
- **CSS3** : Stylage moderne avec flexbox et grid
- **Responsive Design** : Media queries pour tous les appareils

## 🎨 Caractéristiques du design

- **Palette de couleurs** : Dégradé violet/orange sur fond gris clair
- **Typographie** : Segoe UI pour une lisibilité optimale
- **Animations** : Effets de survol et transitions douces
- **Layout** : Grid et flexbox pour une structure flexible

## 📱 Compatibilité

Le site est compatible avec :
- ✅ Chrome (dernière version)
- ✅ Firefox (dernière version)
- ✅ Safari (dernière version)
- ✅ Edge (dernière version)
- ✅ Navigateurs mobiles modernes

## 🚀 Comment utiliser

1. Clonez ce repository :
   ```bash
   git clone https://github.com/[votre-username]/hst-vabh-nah.git
   ```

2. Naviguez dans le dossier du projet :
   ```bash
   cd hst-vabh-nah
   ```

3. Ouvrez `index.html` dans votre navigateur web :
   ```bash
   # Double-cliquez sur index.html ou utilisez :
   start index.html  # Windows
   open index.html   # macOS
   ```

## 🔧 Personnalisation

### Modifier les couleurs
Les couleurs principales sont définies dans `style.css` :
```css
/* Couleurs du dégradé principal */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### Modifier le contenu
- **Texte** : Modifiez directement dans `index.html`
- **Styles** : Personnalisez dans `style.css`
- **Images** : Ajoutez des balises `<img>` dans les sections appropriées
